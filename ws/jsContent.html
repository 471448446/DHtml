javascript 使用
script 标签
HTML 中的脚本必须位于 < script> 与 < /script> 标签之间。
脚本可被放置在 HTML 页面的 <body> 和 <head> 部分中。 script标签，会告诉 JavaScript 在何处开始和结束，包含JS代码。
JavaScript 是所有现代浏览器以及 HTML5 中的默认脚本语言。
函数与事件
如果我们把 JavaScript 代码放入函数中，就可以在事件发生时调用该函数。
<body> 和 <head> 中的 JavaScript
您可以在 HTML 文档中放入不限数量的脚本。通常的做法是把函数放入<head> 部分中，或者放在页面底部。这样就可以把它们安置到同一处位置，不会干扰页面的内容。

Js 变量
命名：
  • 变量必须以字母开头
  • 变量也能以 $ 和 _ 符号开头（不过我们不推荐这么做）
  • 变量名称对大小写敏感（y 和 Y 是不同的变量）
类型：
var 
       var a=10;
       var a=10,b=20;
重新声明 JavaScript 变量,变量的值不会变化。 
       var carname="Volvo";
       var carname;
var的值还是carname。
JavaScript 数据类型
动态数据类型，这个于java，c++面向对象的数据类型不一样。
var x; // x 为 undefined
var x = 6; // x 为数字
var x = "Bill"; // x 为字符串
所以说基本的数据类型都用var来表示
数组：
       var arrays=new Array();
       var arays=new Array("A","B");
       var arr=["a","b"];
对象：
这个有点区别。
对象由花括号分隔。在括号内部，对象的属性以名称和值对的形式 (name : value) 来定义。属性由逗号分隔
       var person={firstname:"Bill", lastname:"Gates", id:5566};
上面例子中的对象 (person) 有三个属性：firstname、lastname 以及id。.
Undefined 和 Null
undefined:表示么有值。值就是undefines就是没有初始化。
null:表示赋值为空。
